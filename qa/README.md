# Q&A Knowledge Base

Detailed answers to questions raised in GitHub issues and PRs.

Each file is named `<issue-or-pr-number>-<short-slug>.md` and contains:
- The original question (with link to the issue/PR)
- A detailed answer with citations to survey papers
- Links to relevant `notes/*.md` entries

A brief 1–2 sentence summary of each answer is also added inline to the
relevant document (`insights.md` or `notes/*.md`), linking back to the
issue/PR.
